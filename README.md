# Swiggy Group Ordering & Split Bill Feature
### PM Portfolio Project — PRD + Working Prototype

---

## 🧩 What is this?
A product concept and working prototype for a **Group Ordering feature on Swiggy** — where friends can order food together from the same restaurant and each person pays for their own items. No more one person paying for everyone and chasing others for money.

---

## 🎯 Problem Statement
When friends or coworkers want to order food together on Swiggy, there's no shared ordering session. One person ends up placing the full order, paying the entire amount, and then manually collecting money from others via UPI or WhatsApp. This is frustrating and kills the group ordering experience.

---

## 📱 Prototype Screens
| Screen | Description |
|--------|-------------|
| 🏠 Home Screen | Feature intro with CTA to start or join a group order |
| 👥 Create Group Screen | Set group name, restaurant, time limit and generate shareable link |
| 🛒 Split Bill Screen | Real-time view of each member's items and individual amounts |
| ✅ Confirm Order Screen | Final payment summary before placing the group order |
